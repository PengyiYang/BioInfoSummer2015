Practice 1

Replicate the t-test result above without using the R built-in function ‘t.test’.

Definition of two-sample t-test is as follows:

\(H_0\): \(\mu_1 = \mu_2\)

\(H_1\): \(\mu_1 \neq \mu_2\)

Test statistic:

\(T=\frac{\overline{Y}_1 - \overline{Y}_2}{\sqrt{s_1^2/N_1 + s_2^2/N_2}}\)
